# Task-18-JavaScript-DOM-and-Event-Handling

A simple beginner-friendly JavaScript project demonstrating **DOM Manipulation** and **Event Handling**. This application allows users to interact with color boxes and update a greeting message dynamically using JavaScript.

---

## Objective

The objective of this project is to understand how JavaScript interacts with HTML elements using the DOM and how user events can be handled to create interactive web pages.

---

## Features

*  Click on a color box to change its background color.
*  Only the selected box changes color.
*  Enter your name and display a personalized greeting.
*  DOM manipulation using JavaScript.
*  Event handling with button click events.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM & Events)
